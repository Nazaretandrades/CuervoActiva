# TFG-CuervoActiva
Proyecto Integrado - 2DAM

CuervoActiva es una aplicación web y móvil para la gestión de eventos y actividades culturales de El Cuervo de Sevilla. Permite a los usuarios registrarse, consultar eventos, marcar favoritos, visualizar un calendario y más. La app está diseñada con un enfoque mobile first y es responsive para web.
